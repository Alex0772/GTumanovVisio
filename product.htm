Продукты Visio

2014 год. 8 апреля 2014 года завершилась поддержка Office 2003.

Продолжается поддержка Visio 2007, Visio 2010, Visio 2013. Это основные применяемые версии. Visio 2003 становится раритетом. Visio 2010 исчезает из продаж, вытесняясь Visio 2013.

Актуальными становятся вопросы "стоит ли и когда стоит переходить на Visio 2013" и "какую версию Visio 2013 предпочесть".

  • Microsoft Visio 2007 Standard, Microsoft Visio 2007 Professional.
  • Microsoft Visio 2010 Standard, Microsoft Visio 2010 Professional, Microsoft Visio 2010 Premium.
  • Microsoft Visio 2013 Standard, Microsoft Visio 2013 Professional, Microsoft Visio 2013 for Office 365.

Сравнение версий Visio 2007 между собой довольно полно отражено на http://support.microsoft.com/kb/896660. Только надо помнить, что в разделе "Сравнение схем..." там идет сравнение номенклатуры поставляемых трафаретов.

Изменения, внесенные в Visio 2010 по сравнению с Visio 2007, и сведения по выполнению миграции описаны на http://technet.microsoft.com/ru-ru/library/cc179128.aspx. Основные новшества в изменении интерфейса: появление пользовательского интерфейса Fluent, ленты, представления BackStage. В работе с ShapeSheet появился интерфейс IntelliSense. Его основные особенности: поиск по ключевым слова, автозавершение ключевых слов, всплывающее окно определения ключевого слова, всплывающее окно подсказок к сигнатуре функции, проверка скобок, ввод многострочных формул для разработчиков, поддержка локальных ссылок и ссылок на другие таблицы.

Более мелкие изменения коснулись строки состояния, настройки рукописного ввода, средства поиска шейпов, расположения окна трафаретов.

Немного изменились дополнения. Дополнение "Цвет по значению" заменено дополнением "Рисунки, связанные с данными". Инструмент "ShapeStudio" удален. Проекты надстроек, которые раньше поставлялись в составе SDK Visio, перенесена непосредственно в Visual Studio 2010.

Изменения, внесенные в Visio 2013 по сравнению с Visio 2010, описаны на http://office.microsoft.com/ru-ru/visio-help/HA102749364.aspx. В описании, правда, встречаются рекламные приемы... Так, например, фразу "шаблон UML стал более удобным" следует читать как "дополнение для работы с UML удалено, теперь остался только трафарет".

Самое главное изменение - это, пожалуй, изменение формата файлов. Документы Visio теперь представляют собой архивированный пакет XML-файлов. С одной стороны, такой шаг нарушает совместимость с предыдущими версиями. С другой, формат документов становится более открытым. Вплоть до того, что можно строить свои приложения для извлечения данных и просмотра документов, не имея Visio.

Довольно много изменений дизайнерского плана: темы с вариантами, специальные эффекты для шейпов, улучшение шаблонов и трафаретов.

Существенны такие изменения, как: возможность добавления изображений ко всем сотрудникам на оргсхеме одновременно, возможность копирования страниц, возможность совместной одновременной работы с документом.

Посмотреть отличия версий Visio 2013 между собой можно на https://visioport.ru/biblio/st116.htm.

Интересным также может оказаться сравнение набора поставляемых трафаретов для версий Standard и Professional. Из этого сравнения видно, что в младшей версии урезана не только функциональность, но и существенно сокращен набор трафаретов. Без сокращения идут только короткие наборы General и Schedule, остальные урезаны. И совсем удалены наборы Engineering, Software and Database.

Сравнение версий 2007, 2010 и 2013 можно посмотреть на http://office.microsoft.com/ru-ru/visio/FX103791748.aspx
