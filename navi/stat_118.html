Текст

Chris Roth в статье Top Twelve Text Tips выделил 12 главных советов при работе с текстом шейпов. Знакомство с ними снимает множество вопросов, которые возникают у новичков.

  • Использование инструмента "Текстовый блок" для изменения положения текста относительно шейпа
  • Использование F2 для перехода в режим редактирования текста
  • Использование двойного щелчка для редактирования текста
  • Связывание полей данных с текстом шейпа
  • Компенсация наклона шейпа
  • Размер текстового блока
  • Использование текста шейпа в качестве входных данных
  • Сравнение строк в ShapeSheet
  • Использование GRAVITY для наклона текста
  • Манипулирование списками значений с помощью функций LOOKUP и INDEX
  • Синхронизация размера шрифта с размером шейпа
  • Перемещение текста с помощью управляющей точки
