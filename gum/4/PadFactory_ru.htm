PadFactory - утилита для создания ИЕТР в Visio

en

PadFactory - утилита для создания простейших интерактивных технических руководств на основе Visio. PadFactory позволяет встроить объект WordPad в шейпы или мастер-шейпы.

Утилита выполнена в виде документа Visio, содержащего макросы.

 Загрузить с этого сайта.

Что такое ИЭТР (Интерактивное Электронное Техническое Руководство) в простейшем случае? Это иерархический документ, в котором каждый блок может быть раскрыт для показа уточняющей информации.

PadFactory - это инструмент для создания таких руководств. Он подготавливает шейпы специальным образом, чтобы шейп мог содержать статью или раздел руководства. По умолчанию пользователь будет видеть только диаграмму. Но щелкнув по шейпу, он увидит окно с дополнительным текстом. И все это находится в едином файле Visio.

PadFactory вставляет объект WordPad в произвольный шейп или мастер-шейп.

Шейп превращается в группу, в которой к исходному объекту добавляется встроенный. Двойной щелчок по такому шейпу открывает встроенный объект для редактирования или просмотра.

Приложение выполнено в виде документа Visio, содержащего несколько макросов. Макросы вызываются через контекстное меню страницы.
